    LIST TAG

    • List tag allow us to group a set of related items in lists.
    • Basically it is having three types :
      1. Unordered list
      2. Ordered list
      3. Description List

    Unordered list

  • < ul> Defines an unordered list
  • < li> Defines a list item
  • An unordered list starts with the < ul> tag. Each list item starts with the < li> tag.
  • The list items will be marked with bullets (small black circles or disc) by default.
  • We are having one attribute for the unordered tag is type="".
  • By using the type attribute we can change the default bullets listing to Square and circle.

    • Syntax : < ul>< li>This is default listing.< /li>< /ul>

    • This is default listing.

    Syntax : < ul type="square">< li>This listing by using square type attribute.< /li>< /ul>

    • This listing by using square type attribute.

    Syntax : < ul type="circle">< li>This listing by using circle type attribute.< /li>< /ul>

    • This listing by using circle type attribute.

    Ordered list

  • < ol> Defines an ordered list
  • < li> Defines a list item
  • An ordered list starts with the < ol> tag. Each list item starts with the < li> tag.
  • The list items will be displayed in particular order with numbers by default.
  • We are having two attributes for the ordered tag is type="" & start="".
  • By using the type attribute we can change the default number listing to alphabatic and roman numbers.

    • Syntax : < ol>< li>This is default listing.< /li>< /ol>

    1. This is default listing.

    Syntax : < ol type="a">< li>This listing by using alphabatic type attribute.< /li>< /ol>

    1. This listing by using alphabatic type attribute.

    Syntax : < ol type="A">< li>This listing by using alphabatic type attribute.< /li>< /ol>

    1. This listing by using alphabatic type attribute.

    Syntax : < ol type="i">< li>This listing by using roman number type attribute.< /li>< /ol>

    1. This listing by using alphabatic type attribute.

    Syntax : < ol type="I">< li>This listing by using roman number type attribute.< /li>< /ol>

    1. This listing by using alphabatic type attribute.

  • By using start attribute we can put starting index for ordered list,we need to do it explicitly.
  • The start attribute will always take no as a input for all type of ordered listing.

    • Syntax : < ol type="I" start="5">< li>item< /li>< li>item< /li>< li>item< /li>< li>item< /li>< li>item< /li>< /ol>

    1. item
    2. item
    3. item
    4. item
    5. item

    Description List

  • < dl> Defines a description list
  • < dt> Defines a term in a description list
  • < dd> Describes the term in a description list
  • A description list is a list of terms, with a description of each term.
  • The < dl> tag defines the description list, the < dt> tag defines the term (name), and the < dd> tag describes each term.

    • Syntax : < dl>
    < dt>Coffee< /dt>
    < dd>- black hot drink< /dd>
    < dt>Milk< /dt>
    < dd>- white cold drink< /dd>

    Coffee
    - black hot drink
    Milk
    - white cold drink
