    CHARACTER FORMATING TAGS

    Bold tag

    < b>- Bold text

    The HTML < b> element defines bold text, without any extra importance.
    Sysntax : < b> Bold < /b>
    The text is Bold.

    < strong> - Important text

    The HTML < strong> element defines text with strong importance. The content inside is typically displayed in bold.
    Sysntax : < strong> Important text < /strong>
    The text is Important text .

    Italic tag

    < i> - Italic text

    The HTML < i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.
    The < i> tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.
    Sysntax : < i>Italic text< /i>
    The text is Italic text.

    < em> - Emphasized text

    The HTML < em> element defines emphasized text. The content inside is typically displayed in italic.
    A screen reader will pronounce the words in < em> with an emphasis, using verbal stress.
    Sysntax : < em>Emphasized text< /em>
    The text is Emphasized text.

    Underline tag

    < u> - Underline text

    It is < u> element used to underline the test in the web pages.
    Sysntax : < u>Underline text
    The text is Underline text.

    < ins> - Inserted text

    The HTML < ins> element defines a text that has been inserted into a document. Browsers will usually underline inserted text.
    Sysntax : < ins>Inserted text< /ins>
    The text is Inserted text.

    Big text tag

    < big> - Big text

    The HTML < big> used to display big text then the normal text on the web page.
    Sysntax : < big>Big text< /big>
    The text is Big text.


    Small tag

    < small> - Smaller text

    The HTML < small> element defines smaller text.
    Sysntax : < small>small text< /small>
    The text is small text.


    Delete tag

    < del> - Deleted text

    The HTML < del> element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.
    Sysntax : < del>Deleted text< /del>
    The text is Deleted text.


    < strike> - strike text

    The HTML < strike> element defines striked text in the web pages when some word is replaced instead of that word.
    Sysntax : < strike>strike text< /strike>
    The text is strike text.


    Superscript tag

    < sup> - Superscript text

    The HTML < sup> element defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like (a+b)2.
    Sysntax : < sup>superscript text< /sup>
    The text is superscript text.


    Subscript tag

    < sub> - Subscript text

    The HTML < sub> element defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O.
    Sysntax : < sub>Subscript text< /sub>
    The text is Subscript text.
