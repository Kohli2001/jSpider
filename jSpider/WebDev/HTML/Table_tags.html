    TABLE TAG

    • html uses table tag to represent the data in the form of rows and columns on the web page.
    • The rows for the table are creates using the tr tag.
    • The th tag is used to dispaly the heading the heading for the table.
    • The text written inside the th tag is bold by default.
    • The data for the heading is written within the td tag.
    • We are having one attribute for giving the border to the table is border

    Sysntax :
    < table border="">
    < tr>
    < th>.....< /th>
    < /tr>
    < tr>
    < td>.....< /td>
    < /tr>
    < /table>


    Attributes

    • Basically we are having two attributes for th and td tag that are Colspan and rowspan.
    • Colspan and rowspan are the attributes which allows occupy single cell to occupy more than one row or column.
    • When we use colspan then by default it will occupy only one column.

      • Sysntax : < th colspan="No. of columns should be occupied">A< /th>

    • We can change it according to use.

      • EXAMPLE :-
      A B
      c d e f
      g h i j

      In the above example we have used colspan="2" for th A & B so it is occuping two columns


    • When we use rowspan then by default it will occupy only one row.

      • Sysntax : < th rowspan="No. of rows should be occupied">A< /th>

    • We can change it according to use.

      • EXAMPLE :-
      A C D
      E F
      B G H
      I J
