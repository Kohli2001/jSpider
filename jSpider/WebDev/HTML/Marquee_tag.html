    Marquee Tag

    The HTML < marquee> tag is used for scrolling piece of text or image displayed either horizontally across or vertically down your web site page depending on the settings.

    Syntax

    < marquee behavior="" direction=""> < /marquee>

    Example

    < !DOCTYPE html>
    < html>
    < head>
    < title>HTML marquee Tag< /title>
    < /head>
    < body>
    < marquee>This is basic example of marquee< /marquee>
    < marquee direction="up">The direction of text will be from bottom to top.< /marquee>
    < /body>
    < /html>

    Specific Attributes

    The HTML < marquee> tag also supports the following additional attributes

    1) Direction : it specifies the direction in which the contents is going to Scroll
    Note>>> the default direction is LEFT

    2) ScrollAmount: it controls the speed of the scroll of an elements
    Note>> higher the scrollamount value the speed of the element will increase
    3) Behavour: it specifies the type of scroll of an element
